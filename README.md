# react-stuff
Repositório para o curso de React da FrontendMasters. 

## Conceitos Aplicados

 - State
 - Props
 - Hooks
 - Router
 - Class Components
 - Error Boundaries
 - Context
 - Portals

## Instalação

**Pré-requisitos: Yarn e acesso à Internet.**

Clone este repositório, abra um terminal e navegue até a pasta ``adopt-me``. Execute o comando ``yarn dev``.
